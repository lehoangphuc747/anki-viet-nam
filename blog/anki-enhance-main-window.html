Enhance Main Window – Plugin hiển thị thông tin chi tiết thẻ Anki

11 tháng 6, 2025 · 2 phút để đọc

Giới thiệu về Enhance Main Window​

Plugin Enhance Main Window là một công cụ mạnh mẽ cho phép bạn thêm các cột dữ liệu vào cửa sổ chính của Anki. Điều này giúp bạn có cái nhìn chi tiết hơn về từng thẻ học, bao gồm:

  • Ease factor (hệ số dễ dàng)
  • Lapses (số lần quên)
  • Number of reviews (số lần ôn tập)
  • Card type (loại thẻ)
  • Deck name (tên bộ thẻ)
  • Tags (nhãn)

Nguồn gốc​

Bộ template của plugin này được phát triển dựa trên mã nguồn từ Reddit.

Ngoài ra, Shige cũng đã phát triển một phiên bản nâng cấp với nhiều tính năng hơn: 📊Enhance main window - add many columns to home decks (Customized by Shigeඞ)

Hướng dẫn cài đặt​

  1. Mở Anki
  2. Vào Tools -> Addons
  3. Tìm Enhance Main Windows
  4. Nhấn vào Config
  5. Sao chép và dán mã cấu hình
  6. Nhấn Save
  7. Khởi động lại Anki

Cài đặt nhanh​

  • Tải plugin trên AnkiWeb: Enhance Main Window (877182321)
  • Add-on code: 877182321
